# Evolvex team wiki
Team wiki.

## Getting Started

### Install Deps

```bash
npm i
```

### Setup .env

```bash
cp .env.sample .env
```

change the values.

### Prisma setup

```bash
npx primsa migrate
```

### Run the server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
